# Project Notes

## Source-status rule

Clavicula Lumenosi separates several layers:

1. **Historical / source layer** — edition or manuscript witnesses, where available.
2. **Analytical Fource layer** — modern interpretive mapping.
3. **Generated glyph layer** — modern generated symbols or placeholders.
4. **Quarantine layer** — coercive, destructive, overclaimed, or unverified material.
5. **Closure layer** — consent, boundary, grounding, and return.

## Greater Key Pentacles

The current model contains a working ledger for the 44 Greater Key pentacles. Some plate/crop layers may rely on remote source links or browser crop panes. Hand-cut local crop assets are a future archival step.

## Akashic Coherence Ledger

The Akashic Coherence Ledger is a disciplined trace-archive model. It does **not** claim access to a literal omniscient cosmic database.

Core rule:

> Everything leaves a trace, but no trace is the whole.

## Safety / interpretation

This project is best read as symbolic, historical, philosophical, and interface-design work. It should not be used as an instruction set for coercive ritual practice, domination, self-harm, or unsafe experimentation.
